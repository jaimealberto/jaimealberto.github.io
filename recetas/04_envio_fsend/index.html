Firefox Send, envio archivos

JaimeAlberto

Subir archivo servicio Firefox Send con password enviando link mediante mail

Paquetes necesarios:

  • ffsend
  • SendEmail

Ejemplo:

ffsend u <FILE_NAME> --password > test && awk '{print $3}' test > link.txt && rm -rf test && sendemail -f <MAIL_FROM> -t <MAIL_TO> -s smtp.gmail.com:587 -o tls=yes -xu <USER_MAIL_FROM> -u "Link descarga" -o message-file=link.txt && rm -rf link.txt && clear
-i trabajara de incógnito ffsend, no deja historial.
-xp cuando se utiliza SendEmail se puede especificar la password de quien envía el correo, no es aconsejable.

Al ejecutar pide dos password, la primera para el archivo que no se ve. La segunda es la del correo que se ve en el terminal. Limitaciones: numero máximo de descargas 1, duración del almacenamiento 24 horas tamaño máximo por archivo 2 GB.

Subir un archivo al servicio Firefox Send con password y enviar el link a un contacto de Telegram

Paquetes necesarios:

  • ffsend
  • Telegram-cli

Ejemplo:

ffsend u <FILE_NAME> --password > test && awk '{print $3}' test > link.txt && rm -rf test && 
telegram-cli -W -e "msg <CONTACT_NAME> hello from the command line"
telegram-cli -W -e "send_document <CONTACT_NAME>C <PATH_TO_FILE>"
