Donde están mis contraseñas Wifi

Muchas veces estamos trabajando con un equipo que tenemos ya configuradas las conexiones a varias redes Wifi y necesitamos ver una configuración para utilizarla en otro equipo o dispositivo, pero que pasa cuando no podemos ver por la interface grafica la contraseña de conexión. Pues bien a continuación vamos a ver como solucionar esto.

Linux

Abrimos un terminal y vamos al lio:

sudo cat /etc/NetworkManager/system-connections/nombre_de_la_red_wifi | grep psk=

MacOs

Abrimos un terminal y realizamos los siguientes pasos:

Identificamos interfaces de red Wifi:

security find-generic-password -wa red_wifi

Listamos los perfiles Wifi :

networksetup -listpreferredwirelessnetworks <interface>

Consultamos la contraseña del perfil:

security  find-generic-password -wa <perfil>

Aparece una ventana solicitando usuario y contraseña con derechos de administrador luego mostrara la contraseña.

Windows

Abrimos nuestro terminal de Windows y realizamos los siguientes pasos:

Listamos los perfiles Wifi

netsh wlan show profile

Consultamos la contraseña del perfil:

netsh wlan sho profile name=nombredelperfil key=clear

Esto nos muestra una salida con una linea que contiene “Contenido de la clave : Esta es la contraseña “.
