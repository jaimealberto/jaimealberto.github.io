Combinación de teclas

Alguna de las combinaciones de teclas que utilizaremos el día a día cuando escribamos nuestro código o nuestros documentos utilizando VSCode. Estas son las mas habituales:

Cortar ⌘ + X

Borrar ⌘ + ⌫

Pegar ⌘ + V

Deshacer ⌘ + Z

Seleccionar todo ⌘ + A

⌘ Comando (cmd)

MultiCursor en VSCode

Una de las utilidades muy interesantes de cualquier editor de código es el multicursor, en VSCode se activa con la siguiente combinación de teclas:

⌥ + ⌘ + ↑/↓

⌥ Opción (alt) ⌘ Comando (cmd) ↑↓ Flecha arriba, Flecha abajo

Una vez realizadas todas las operaciones necesarias con el multicursor ton pulsar la tecla esc se desactivara.

Brew, copia seguridad

Alguna que otra vez, rara vez, tendremos que reinstalar el sistema operativo o bien en un equipo nuevo queremos instalar todos los paquetes brew que tememos en nuestro equipo actual. Esto se conoce comunalmente como “ Dumping Hombrew packages “. Ahora veremos como hacerlo:

Desde el terminal del equipo que tenemos todo instalado y funcionando:

brew bundle dump

Esto crea un fichero de texto llamado “ Brewfile “, el cual contiene de una manera estructurada todo lo instalado en el equipo mediante brew. Lo copiamos al equipo nuevo. Desde el terminal del equipo nuevo donde tengamos copiado el fichero tecleamos:

brew bundle

Y ya instalara todo automáticamente.

Acuse de recibo en Mail

Cierto es que una de las características que no tiene el gestor de correo por excelencia de Mac, Mail, es el acuse de recibo. A veces es una característica que muchos añoramos, pero no por ello tenemos que plantearnos utilizar otro gestor de correo. Tenemos dos opciones de hacer esto. Desde mi querida terminal o utilizando alguna de las aplicaciones que hace lo mismo pero desde el entorno gráfico. A continuación veremos como hacerlo desde el terminal.

Cerramos Mail y abrimos nuestro terminal, tecleamos la siguiente linea de comando:

defaults write com.apple.mail UserHeaders ‘{”Disposition-Notification-To”=”tucorreo@dominio.com”;}

“tucorreo@dominio.com” Es la dirección de correo donde queremos que nos envíen el acuse de entrega del mail.

Abrirnos Mail. Ahora todos los correos que enviemos solicitaran acuse de recibo a la dirección de correo indicada. Si esto no funcionara repite el proceso cercioran dote de haber escrito bien las comillas, comillas simples, paréntesis, etc. Y recuerda que el acuse de recibo tarda en llegar.

Para desactivarlo, cerramos Mail. Abrimos nuestro terminal, si lo hemos cerrado, si no vale el mismo que teníamos abierto y tecleamos la siguiente liena de comando:

defaults delete com.apple.mail UserHeaders

Abrimos Mail y todo seguirá como antes.
