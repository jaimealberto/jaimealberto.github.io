Todo el que me conoce sabe que no soy un fan de Windows. Si tengo que elegir como entorno de trabajo mi ultima opción siempre sera Windows. Tampoco soy ningún nazi de los que solo defienden un sistema operativo a muerte, creo que cada entorno tiene sus cosas buenas y sus cosas malas. En ocasiones no tengo mas remedio que trabajar con Windows, con lo cual voy a dedicar algunos post para tener como referencia de mi día a día y ser lo mas productivo posible.

Como el cualquier entorno grafico es muy productivo saber cuales son las teclas rápidas para organizar las ventas, cambiar de ventana en ventana o entre escritorio. Así que vamos a ver como se hace en Windows. Vamos al lio.

Atajos de teclado

⌥       Alternativa (alt)
⇥       Tabulador (Tab)
WIN     Tecla Windows ()
← → ↑ ↓ Flechas

ALT + TAB Nos permite movernos rápidamente por todas las ventanas abiertas. Manteniendo pulsada Alt y cada vez que pulsemos Tab vamos cambiando de ventana.

WIN + TAB Historial actividad Winodws10.

WIN + FLECHAS Si combinamos la tecla de Windows con las flechas podemos realizar las siguientes acciones:

WIN + ↓ minimiza la ventana activa.
WIN + ↑ maximiza la ventana activa.
WIN + → coloca la ventana activa en la parte derecha de la pantalla.
WIN + ← coloca la ventana activa en la parte izquierda de la pantalla.

WIN + M Minimiza todas las ventanas.

Tipos de vista

Windows es capaz de organizar de forma automática las ventanas sin tener que hacer nada para poder tenerlas siempre a la vista. Solo tenemos que hacer clic con el botón derecho sobre la barra de tareas del y elegir una de las opciones disponibles:

Mostrar ventanas en cascada
Mostrar ventanas apiladas
Mostrar ventanas en paralelo
Mostrar escritorio

Acoplar ventanas (Hot Corners)

Otra forma de organizar y colocar las ventanas en Windows es arrastrándolas directamente a las esquinas de la pantalla. Según donde coloquemos la ventana tenemos 8 formas diferentes de hacerlo:

Arriba, abajo, izquierda, derecha: las ventanas que coloquemos en los laterales ocuparán todo el ancho o largo (según el lateral elegido) y la mitad del contrario.
Esquinas: nos permite colocar la ventana directamente en la esquina seleccionada, ocupando la mitad tanto verticalmente como horizontalmente. De esta forma podemos tener hasta 4 ventanas a la vez en perfecta armonía.

Escritorios virtuales

Los escritorios virtuales en otros sistemas operativos es algo habitual hace ya años, pero en Windows de manera nativa sin software adicional esta disponible a partir de su versión 10. Esto nos permite disponer de varios escritorios, colocando en cada uno los programas y las ventanas que queramos de esta manera podemos tener varios espacios de trabajo dedicando cada uno a cosas en concreto.

Crear un escritorio virtual

Pulsamos WIN + Tab, esto nos muestra la línea de tiempo en la parte superior. Hacemos clic en “ Nuevo escritorio ”.

Cambiar el nombre a un escritorio virtual

Por defecto, los escritorios se llaman “ Escritorio x “ a todos los escritorios que vayamos creando. Si queremos ser más organizados podemos ponerle el nombre que queramos a cada uno de ellos. Hacer clic sobre el nombre del escritorio virtual y así podremos ver el cuadro de texto que nos dejará cambiar el nombre. Opción estará disponible para todos los usuarios a partir de Windows 10 20H1.

Cerrar los escritorios virtuales

Colocamos el ratón sobre él y en la parte superior derecha de su miniatura veremos la típica X de cerrar. Hacemos clic sobre la X y automáticamente se cerrara. Si lo cerramos por accidente y tenemos ventanas abiertas, estas pasarán al escritorio principal.

Movernos entre escritorios virtuales

⇥ + WIN + ( ← “o” → ) Para movernos entre todos los escritorios que tengamos.

Agitar una ventana

Seleccionamos la una ventana que queramos en primer plano por la barra superior y, manteniendo el botón derecho del raton apretado, la agitamos. Esto minizara el resto de ventnas a la vez y dejará en primer plano la que hemos agitado.

Programas para organizar ventanas

Tenemos también disponibles una serie de programas diseñados para ayudarnos a organizar mucho mejor las ventanas de nuestro Windows. Aquí dejo una lista por si interesa:

  • Microsoft PowerToys (FancyZones)
  • Shutter Split Screen
  • AquaSnap

Snaps en Windows 11

Windows 11 trae de forma nativa una función para la que, hasta ahora, era necesario utilizar software de terceros. Los Snaps llevan directamente las FancyZones de las PowerToys al explorador de Windows.

Pulsando WIN + Z, o colocando el ratón sobre el botón maximizar, podremos ver los diseños disponibles.

Problemas de espacio, escalar

Por muy bueno que sea el programa que usemos para organizar nuestras ventanas, siempre estaremos limitados al tamaño de nuestro monitor. Si usamos un monitor relativamente pequeño, tendremos poco espacio para colocar las ventanas abiertas. Podemos obtar por cambiar la escala de Windows desde el menú de Configuración, en el apartado Sistema > Pantalla. Aquí dispnemos apartado llamado «Escala y distribución». Cuando más pequeño sea el valor de la escala más pequeños serán los elementos, podremos colocar más ventanas en el monitor. Si esto no es suficiente, nos toca comprar un monitor mas grande.
