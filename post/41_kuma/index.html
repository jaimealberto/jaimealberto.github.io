Uptime Kuma, simple y eficaz

En muchas ocasiones instalamos y configuramos herramientas las cuales nos sobra el 90 % de las taras que realizan. Muchas veces no necesitamos complicarnos la vida para tener una buena herramienta. Este creo que es el caso de Uptime Kuma.

Es una herramienta de monitorización sencilla la cual su principal objetivo y medir la disponbilidad de servicios y hosts. Sus principales características son:

  • Tiempo de supervisión para HTTP(s) / TCP / HTTP(s) Keyword / HTTP(s) Json Query / Ping / DNS Record / Push / Steam Game Server / Docker Containers.
  • Notificaciones a través de Telegram, Discord, Gotify, Slack, Pushover, Email (SMTP) y ma de 90 servicios de notificacion.
  • Intervalos de 20 segundos.
  • Varios idiomas disponibles.
  • Múlples páginas de estado.
  • Páginas de estado de mapa a dominios específicos.
  • Ping.
  • Información del certificado.
  • Soporta proxy.
  • Soporte 2FA.
