Integrar actualizaciones Visual Studio Code

JaimeAlberto
11-04-2022
Visual Studio Code, DevOps
Índice
  • Debian / Ubuntu
  • RHEL / Fedora
  • MacOs

Visual Studio Code es mi editor preferido por muchos motivos. Pero siempre que toca actualizar el sistema operativo o el software instalado según realizamos su instalación se incluye o no. Pues bien ahora veremos como instalarlo para que al actualizar nuestro sistema se incluya en dicha actualización.

Habitualmente descargamos el paquete de instalación y procedemos a instalar, con lo cual al realizar actualizaciones el VsCode queda desactualizado y cuando lo arrancamos nos avisa, a mi personalmente esto no me gusta nada. Con lo cual vamos añadir métodos para que esto no ocurra. Abrimos nuestro querido terminal en nuestro sistema.

Debian / Ubuntu

Añadimos la llave del repositorio y el repositorio

sudo apt-get install wget gpg
wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > packages.microsoft.gpg
sudo install -o root -g root -m 644 packages.microsoft.gpg /etc/apt/trusted.gpg.d/
sudo sh -c 'echo "deb [arch=amd64,arm64,armhf signed-by=/etc/apt/trusted.gpg.d/packages.microsoft.gpg] https://packages.microsoft.com/repos/code stable main" > /etc/apt/sources.list.d/vscode.list'
rm -f packages.microsoft.gpg

Actualizamos e instalamos

sudo apt install apt-transport-https
sudo apt update
sudo apt install code

RHEL / Fedora

sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/yum.repos.d/vscode.repo'

Actualizamos e instalamos

Utilizando dnf: 

dnf check-update
sudo dnf install code

Utilizando yum: 

yum check-update
sudo yum install code

MacOs

brew install --cask visual-studio-code
